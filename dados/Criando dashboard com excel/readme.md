Dashboard em excel com uma amostra de dados do Xbox Game Pass

A dashboard foi criada com os dados de venda total e com um filtro para as vendas anuais

A primeira informação é um big number, onde mostra o total de vendas da EA Play

A segunda informação, também um big number, mostra o total de vendas do Minecraft Season Pass

A terceira informação, também um big number, mostra o total de descontos aplicados 

A terceira informação é um gráfico de barras onde mostra quantos planos estão em renovação automática ou não

A quarta informação, é um gráfico de barras que mostra o total de vendas do Minecraft Season Pass por cada plano
